Hexo Theme教程

前言

 

在上一篇教程里，我们已经带领大家一步步使用Hexo，在Github Pages上搭建了一个属于自己的技术博客。Hexo的默认主题是Landscape，看起来简洁大方，却是千篇一律。事实上，Hexo有着完善的Theme系统，可以很方便地为你的博客网站更换外观。

本文旨在通过一个简短的博客更换教程，虽不触及Hexo主题的深层原理，但希望让初接触Hexo的读者们了解更换博客主题的基本方法，感受Hexo为我们的博客架设带来的众多便利。

 

我们要做什么？

 

在这篇文章里，我们将一步一步带着大家为自己的博客更换一个主题。本文选择的主题是Icarus，在文章的末尾，我们的博客看起来大概是这个样子。

为了能完成最终效果，你需要一个已经搭建好的Hexo博客，对Hexo博客系统与博客写作有一个基本的概念。

配置主题

 

Icarus有着较为详细的配置文档：

https://blog.zhangruipeng.me/hexo-theme-icarus/tags/Getting-Started/

不仅如此，位于themes/icarus/文件夹下的_config.yml也有着很清晰的注释。

作为例子，我们将带领大家对主题进行一些简单的配置。首先，在主题的默认配置中，三栏信息过于冗余，有喧宾夺主之感，一些信息也与顶栏有所重复。而人阅读顺序一般是从左至右，将一些无关紧要的信息放在左栏反而使内容相形失色。所以我们将把删减后的Widgets全部移到右栏。

除此以外，我们还将修改Profile中的个人信息。

最终效果：

首先，我们需要打开_config.yml，找到widgets字段，将我们不想要的Widgets全部注释，并修改留下的Widgets的position属性为right。

找到profile组件，填入个人信息，略作修改：

widgets:
  \-
    \# Widget name
    type: profile
    \# Where should the widget be placed, left or right
    position: right
    \# Author name to be shown in the profile widget
    author: Yichen Xu
    \# Title of the author to be shown in the profile widget
    \# author_title: Your title
    \# Author's current location to be shown in the profile widget
    \# location: Your location
    \# Path or URL to the avatar to be shown in the profile widget
    avatar:
    \# Email address for the Gravatar to be shown in the profile widget
    gravatar: YOUR_EMAIL_ADDRESS
    \# Whether to show avatar image rounded or square
    avatar_rounded: false
    \# Path or URL for the follow button
    follow_link: 'https://github.com/linyxus'
    \# Links to be shown on the bottom of the profile widget
    \# social_links:
    \#   Github:
    \#     icon: fab fa-github
    \#     url: 'https://github.com/ppoffice'
    \#   Facebook:
    \#     icon: fab fa-facebook
    \#     url: 'https://facebook.com'
    \#   Twitter:
    \#     icon: fab fa-twitter
    \#     url: 'https://twitter.com'
    \#   Dribbble:
    \#     icon: fab fa-dribbble
    \#     url: 'https://dribbble.com'
    \#   RSS:
    \#     icon: fas fa-rss
    \#     url: /

 

此处也可以把avatar_rounded改为true，效果会更好看些：

除此以外，Icarus提供了非常丰富的定制选项，读者们可以借助Icarus的官方文档与_config.yml的注释进行探索。
